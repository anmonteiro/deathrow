## Prerequisites

You will need [Leiningen][1] 2.0.0 or above installed.

[1]: https://github.com/technomancy/leiningen

## Running

Running the following will build the Om Next project:

    lein fig-next

Running the following will build the Om (Now) project:

    lein fig-om
